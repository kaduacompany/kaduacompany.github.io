Pelatihan Web Desain

Ada beberapa hal yang perlu diketahui oleh para pengembang web pemula ketika pertama kali ingin membangun sebuah web dari awal. Agar web yang dibuat nampak profesional perlu memahami beberapa konsep dasar dalam desain web, mulai dari tool yang digunakan, format gambar yang sesuai, hingga pemilihan warna teks dan latar belakang sebuah halaman web.

Program Pelatihan Web Design Tingkat Dasar ini ditujukan bagi pemula dalam bidang website. Materi pelatihan mempelajari konsep dasar desain web, aspek yang perlu diperhatikan dalam membuat sebuah website, web script (HTML) editor, pembuatan layout halaman, link, editing foto, animasi web serta pengetahuan umum internet yang berkaitan dengan website.

Setelah mengikuti pelatihan ini diharapkan peserta memahami konsep dasar desain web, menguasai HTML editor untuk pembuatan desain web dengan memahami aspek yang perlu diperhatikan dalam membuat sebuah website, pengeditan gambar dan animasi sederhana untuk web.
